# Перенос проекта в путь без кириллицы

## Рекомендуемое решение

Кириллица в пути `C:\Всякое\Конструктор сценариев` может вызывать проблемы с Node.js и путями.

## Вариант 1: Перенос в простой путь (рекомендуется)

### Новый путь:
```
C:\scenario-builder
```

### Шаги:
1. Закройте все окна с проектом
2. Скопируйте всю папку проекта в новое место:
   ```cmd
   xcopy /E /I "C:\Всякое\Конструктор сценариев" "C:\scenario-builder"
   ```
3. Или вручную через проводник:
   - Выделите папку "Конструктор сценариев"
   - Скопируйте (Ctrl+C)
   - Перейдите в C:\
   - Вставьте и переименуйте в "scenario-builder"

4. Откройте проект из нового пути:
   ```cmd
   cd C:\scenario-builder
   node server.js
   ```

## Вариант 2: Использовать короткое имя пути (8.3)

Windows создает короткие имена для папок с кириллицей. Можно использовать их:

```cmd
cd C:\VSIKO~1\KONST~1
```

Но это неудобно и ненадежно.

## Вариант 3: Создать символическую ссылку

```cmd
mklink /D C:\scenario-builder "C:\Всякое\Конструктор сценариев"
```

Затем работать через `C:\scenario-builder`

## Рекомендация

**Лучше всего перенести проект в `C:\scenario-builder`** - это решит все проблемы с путями.

После переноса:
1. Откройте проект из нового пути
2. Запустите: `node server.js`
3. Откройте: http://localhost:3000/test-agent.html
