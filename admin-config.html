🛡️ Guardrails Configuration

Определяет, насколько агрессивно guardrails блокируют потенциально опасные шаблоны.
Включите типы рисков, которые вы хотите блокировать. Наведите курсор, чтобы понять, что именно будет блокироваться.
Применяется, если включен риск “Excessive Agency”.
JSON-объект с массивами строк/регулярных выражений. Каждое выражение описывает то, что будет считаться небезопасным. Пример: 
{
  "promptInjection": ["ignore all previous instructions"],
  "insecureOutput": ["rm -rf /"],
  "toolInputDangerous": ["DROP TABLE"]
}
