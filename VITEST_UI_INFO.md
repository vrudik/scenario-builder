# 🧪 Vitest UI - Интерфейс тестирования

## ✅ Тесты запущены с UI интерфейсом

Vitest UI должен автоматически открыться в вашем браузере.

## 📱 Если браузер не открылся автоматически:

### Откройте вручную один из адресов:

- **http://localhost:51204** (стандартный порт Vitest UI)
- Или проверьте вывод в терминале - там будет указан точный адрес

## 🎯 Что вы увидите в Vitest UI:

1. **Список всех тестов** - все тестовые файлы и тесты
2. **Статус выполнения** - какие тесты прошли, какие провалились
3. **Детали ошибок** - если тесты провалились, увидите подробности
4. **Время выполнения** - сколько времени занял каждый тест
5. **Фильтры** - возможность фильтровать тесты по статусу

## 📊 Доступные тесты:

- ✅ `scenario-spec.test.ts` - тесты валидации спецификаций
- ✅ `scenario-builder.test.ts` - тесты компилятора
- ✅ `tool-registry.test.ts` - тесты реестра инструментов
- ✅ `tool-gateway.test.ts` - тесты шлюза инструментов

## 🔄 Обновление

Vitest UI автоматически обновляется при изменении файлов (watch mode).

## ⏹️ Остановка

Нажмите `Ctrl+C` в терминале для остановки тестов.

## 💡 Советы:

- Используйте фильтры для поиска конкретных тестов
- Кликните на тест для просмотра деталей
- Используйте кнопку "Run All" для повторного запуска всех тестов
