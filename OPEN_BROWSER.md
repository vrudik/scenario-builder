# 🌐 Веб-интерфейс запущен!

## ✅ Сервер работает

Веб-сервер запущен и готов к использованию.

## 📱 Откройте в браузере:

### **http://localhost:3000**

## Что вы увидите:

1. **Dashboard** с карточками всех компонентов:
   - ✅ Scenario Spec
   - ✅ Scenario Builder  
   - ✅ Tool Registry
   - ✅ Tool Gateway
   - ✅ Runtime Orchestrator

2. **Статус системы** - общая информация о компонентах

3. **Информация о тестах** - какие тесты готовы к запуску

4. **Автообновление** - данные обновляются каждые 5 секунд

## 🔄 Обновление данных

Нажмите кнопку "🔄 Обновить данные" для ручного обновления.

## ⏹️ Остановка сервера

Нажмите `Ctrl+C` в терминале, где запущен сервер.

## 🧪 Запуск тестов

В новом терминале выполните:

```bash
npm test
```

Или с UI интерфейсом:

```bash
npm run test:ui
```

Это откроет еще один веб-интерфейс для просмотра результатов тестов.
