Сквозной демо-тест (готовые данные)

Этот экран нужен для быстрой проверки: любой разработчик может загрузить заранее подготовленный сценарий, запустить его одной кнопкой и увидеть ожидаемый результат по шагам.

Инструкция

    Предзаполненный сценарий

    Нажмите «Проверить предзаполненные данные»

    Результат выполнения

    Нажмите «Запустить сквозной тест»
