# Исправление проблемы с браузером

## ✅ Сервер работает!

Сервер запущен и отвечает на порту 3000.

## 🔧 Что делать:

### 1. Откройте страницу через сервер (НЕ через файл!)

**Правильный способ:**
```
http://localhost:3000/test-agent.html
```

**Неправильный способ (не работает):**
- Открытие файла `test-agent.html` напрямую через проводник
- Использование `file://` протокола

### 2. Если страница не открывается:

1. **Проверьте, что сервер запущен:**
   ```cmd
   cd C:\scenario-builder
   node server.cjs
   ```

2. **Откройте в браузере:**
   - http://localhost:3000/test-agent.html
   - http://localhost:3000/dashboard.html
   - http://localhost:3000/

### 3. Если видите ошибки в консоли браузера:

- Нажмите F12 для открытия DevTools
- Перейдите на вкладку "Console"
- Скопируйте ошибки и сообщите

### 4. Проверка статуса:

API работает:
- ✅ http://localhost:3000/api/status
- ✅ http://localhost:3000/api/agent/status

Ollama доступен: ✅

## 🚀 Быстрый запуск:

Дважды кликните на файл:
```
C:\scenario-builder\start-server.bat
```

Затем откройте в браузере:
```
http://localhost:3000/test-agent.html
```
