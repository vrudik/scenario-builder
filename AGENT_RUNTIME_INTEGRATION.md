# Интеграция Agent Runtime с веб-сервером

## ✅ Что сделано:

1. **Создан обработчик Agent Runtime** (`src/web/agent-handler.ts`)
   - Создает Agent Runtime с настройками Ollama
   - Регистрирует базовые инструменты
   - Выполняет запросы пользователей

2. **Создан скрипт выполнения** (`src/web/execute-agent.ts`)
   - Выполняется через `tsx` из `server.cjs`
   - Принимает запрос через временный JSON файл
   - Возвращает результат в stdout

3. **Обновлен server.cjs**
   - Использует `tsx` для выполнения TypeScript модулей
   - Вызывает Agent Runtime для обработки запросов

## 🚀 Как использовать:

1. **Убедитесь, что зависимости установлены:**
   ```cmd
   cd C:\scenario-builder
   npm install
   ```

2. **Убедитесь, что Ollama запущен:**
   ```cmd
   ollama serve
   ```

3. **Запустите сервер:**
   ```cmd
   node server.cjs
   ```

4. **Откройте в браузере:**
   ```
   http://localhost:3000/test-agent.html
   ```

5. **Введите запрос и нажмите "Выполнить"**

## ⚠️ Важно:

- Для работы нужен установленный `tsx` (должен быть в `devDependencies`)
- Ollama должен быть запущен на `http://localhost:11434`
- Модель `llama3.2:1b` должна быть загружена в Ollama

## 🔧 Если что-то не работает:

1. Проверьте, что `tsx` установлен:
   ```cmd
   npx tsx --version
   ```

2. Проверьте, что Ollama доступен:
   ```cmd
   curl http://localhost:11434/api/tags
   ```

3. Проверьте логи сервера в консоли

4. Откройте консоль браузера (F12) для отладки
