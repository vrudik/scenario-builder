# Ручное копирование проекта

## Простой способ (рекомендуется)

### Шаг 1: Откройте проводник Windows

### Шаг 2: Перейдите в папку проекта
```
C:\Всякое\Конструктор сценариев
```

### Шаг 3: Выделите все файлы и папки
- Нажмите `Ctrl+A` чтобы выделить все
- Или выделите вручную нужные файлы

### Шаг 4: Скопируйте
- Нажмите `Ctrl+C` (копировать)

### Шаг 5: Создайте новую папку
- Перейдите в `C:\`
- Создайте папку `scenario-builder`
- Войдите в неё

### Шаг 6: Вставьте
- Нажмите `Ctrl+V` (вставить)

### Шаг 7: Запустите из новой папки

Откройте командную строку (cmd) и выполните:
```cmd
cd C:\scenario-builder
node server.js
```

### Шаг 8: Откройте в браузере
```
http://localhost:3000/test-agent.html
```

## Альтернатива: Использовать robocopy

В командной строке (cmd) выполните:
```cmd
robocopy "C:\Всякое\Конструктор сценариев" "C:\scenario-builder" /E /XD node_modules .git
```

Это скопирует все файлы, исключая node_modules и .git.

## Что копировать

Обязательно скопируйте:
- ✅ Все файлы `.ts`, `.js`, `.json`
- ✅ Папку `src/`
- ✅ Папку `examples/`
- ✅ Папку `tests/`
- ✅ Все `.html` файлы
- ✅ Все `.bat` файлы
- ✅ Все `.md` файлы

Можно не копировать:
- ❌ `node_modules/` (можно переустановить)
- ❌ `.git/` (если используете git)

## После копирования

1. Перейдите в новую папку: `cd C:\scenario-builder`
2. Установите зависимости: `npm install`
3. Запустите сервер: `node server.js`
4. Откройте: http://localhost:3000/test-agent.html
