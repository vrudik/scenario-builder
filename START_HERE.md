## 🚀 Как запустить веб-интерфейс

### Способ 1: Через PowerShell (рекомендуется)

Откройте PowerShell в директории проекта и выполните:

```powershell
.\start-web.ps1
```

### Способ 2: Через npm

```bash
npm run web
```

### Способ 3: Напрямую

```bash
npx tsx src/web/simple-server.ts
```

## 🌐 Откройте браузер

После запуска сервера откройте в браузере:

**http://localhost:3000**

Вы увидите:
- 📊 Статус всех компонентов системы
- ✅ Информацию о тестах
- 🔄 Автообновление каждые 5 секунд

## 🧪 Запуск тестов

В новом терминале:

```bash
# Все тесты
npm test

# С UI интерфейсом (откроется в браузере)
npm run test:ui

# С покрытием кода
npm run test:coverage
```

Полный сервер с БД и админ-API: `node server.cjs` (порт из `PORT`, по умолчанию 3000). Лёгкий режим без Prisma: `npm run web` → `src/web/server.ts`.

## 📝 Примеры использования

```bash
# Пример использования компонентов
npm run example

# Ручной тест
npm run test:manual
```

## ⚠️ Если что-то не работает

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Проверьте порт 3000:**
   ```powershell
   netstat -ano | findstr :3000
   ```

3. **Используйте другой порт:**
   Измените `PORT` в файле `src/web/simple-server.ts`
